### 项目特点
1. 小程序使用流行的微信授权登录，方便快捷。
2. 小程序同时具备发单和接单的功能，无需使用两个小程序。
3. 后端采用采用Spring+SpringMVC+MyBatis的SSM框架，使用IDEA开发，易上手。
4. 数据库使用mysql, 只用4张表便实现了所有基本功能，方便添加新功能。
